This guide shows you how to integrate SeerStack into your application. You'll learn how to authenticate, identify users, and send your first event.

## 1. Get your API Key

You need an API key to send data to SeerStack.
1. Log in to the [SeerStack Dashboard](https://app.seerstack.app).
2. Go to **Settings > Developer**.
3. Copy your **Publishable Key**.

## 2. Install the SDK

<Tabs>
  <Tab title="TypeScript">
    Install the official TypeScript/JavaScript library:

    ```bash
    npm install seerstack
    ```
  </Tab>
  <Tab title="Ruby">
    Add the gem to your Gemfile:

    ```ruby
    gem "seerstack"
    ```

    Then run:

    ```bash
    bundle install
    ```

    Or install directly:

    ```bash
    gem install seerstack
    ```
  </Tab>
</Tabs>

## 3. Initialize the client

<Tabs>
  <Tab title="TypeScript">
    ```typescript
    import SeerStack from 'seerstack';

    const client = new SeerStack({
      apiKey: process.env['SEERSTACK_API_KEY'], // e.g. 'sk_live_...'
    });
    ```
  </Tab>
  <Tab title="Ruby">
    ```ruby
    require "seerstack"

    seerstack = SeerStack::Client.new(
      api_key: ENV["SEERSTACK_API_KEY"] # e.g. 'sk_live_...'
    )
    ```
  </Tab>
</Tabs>

## 4. Send an event

Send a test event to verify your integration.

<Tabs>
  <Tab title="TypeScript">
    ```typescript
    const response = await client.events.capture({
      name: 'test.event',
      user_id: 'internal_user',
      data: {
        message: 'Hello World',
        browser: 'Chrome'
      }
    });

    console.log(response.success); // true
    ```
  </Tab>
  <Tab title="Ruby">
    ```ruby
    response = seerstack.events.capture(
      name: "test.event",
      user_id: "internal_user",
      data: {
        message: "Hello World",
        browser: "Chrome"
      }
    )

    puts(response.success) # true
    ```
  </Tab>
</Tabs>

Check your [Dashboard](https://app.seerstack.app) â€” you should see the event appear instantly.

## 5. Identify a user (optional)

Associate user traits to enrich your events with user context.

<Tabs>
  <Tab title="TypeScript">
    ```typescript
    await client.users.identify({
      user_id: 'user_123',
      email: 'grace@hopper.com',
      name: 'Grace Hopper',
      attributes: {
        plan: 'enterprise',
        team_size: 15
      }
    });
    ```
  </Tab>
  <Tab title="Ruby">
    ```ruby
    seerstack.users.identify(
      user_id: "user_123",
      email: "grace@hopper.com",
      name: "Grace Hopper",
      attributes: {
        plan: "enterprise",
        team_size: 15
      }
    )
    ```
  </Tab>
</Tabs>
